### Цель кейса
Создать десктоп приложение позволяющее администрировать процесс игры в рамках одной партии:
- записывать и проверять корректность ходов
- вычислять промежуточные результаты
- поддерживать инвариантность
- хранить историю игр
- вычислять рейтинги и статистики по игрокам

### О выбранной игре
Выбрана игра **Морской бой**, но с двумя дополнительными режимами
- **режим "Experienced"** — количество ходов текущего игрока равно количеству его непотопленных кораблей
- **режим "Extended"** — возможность выбрать дополнительное вооружение: самолёты и мины

### Сборка и запуск
```bash
./gradlew run      # запуск приложения
./gradlew test     # запуск тестов
./gradlew build    # сборка проекта и запуск тестов
```
